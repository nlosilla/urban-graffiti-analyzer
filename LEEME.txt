ETIQUETDOR DE GRAFITTIS

Este repositorio contiene una herramienta interactiva en Python diseñada para la selección y clasificación manual de grafitis mediante recortes realizados directamente en imágenes digitales. El proyecto está orientado a investigaciones en etnografía urbana, arqueología del presente, análisis visual y estudios culturales que requieren extraer información estructurada a partir de fotografías.

CARACTERÍSTICAS

-Interfaz interactiva basada en OpenCV.
-Selección de áreas mediante arrastre del ratón.
-Clasificación inmediata usando el teclado.
-Generación automática de nombres de archivo para cada recorte.
-Registro de identificadores y coordenadas asociados al recorte.
-Exportación final en un archivo Excel con todos los datos generados.
-Sistema de configuración independiente mediante config.json.
-Compatibilidad con Windows, macOS y Linux.

ESTRUCTURA DEL PROYECTO

El proyecto debe organizarse de la siguiente manera:
-etiquetador_grafitis.py: archivo principal del programa.
-config.json: archivo que contiene las rutas necesarias para la ejecución.
-datos_grafitis.csv: archivo que incluye ID, coordenadas X/Y y nombre de la fotografía.
-Carpeta input_images: contiene las imágenes originales.
-Carpeta output_crops: almacena los recortes generados.

REQUISITOS PREVIOS

Para ejecutar correctamente la herramienta es necesario contar con:
-Python 3.8 o superior.
-Dependencias instaladas: OpenCV, Pandas y Pillow.
-Un archivo CSV con los datos mínimos requeridos.
-Un archivo de configuración válido.

INSTALACIÓN

-Descargar o clonar el repositorio en una carpeta local.
-Crear y activar un entorno virtual (opcional pero recomendado).
-Instalar las dependencias mediante:
-pip install -r requirements.txt
-Verificar que las carpetas input_images y output_crops existen y contienen la información esperada.
-Configurar las rutas en el archivo config.json.

CONFIGURACIÓN

El archivo config.json debe incluir las rutas necesarias para que el programa pueda acceder a las imágenes, al archivo CSV y a la carpeta donde se guardarán los recortes. Un ejemplo de configuración es el siguiente:

{
  "input_images": "input_images",
  "output_crops": "output_crops",
  "input_csv": "datos_grafitis.csv"
}

USO DEL PROGRAMA

-Ejecutar el script desde la carpeta raíz del proyecto mediante:
	python etiquetador_grafitis.py
-Seleccionar un área en la imagen arrastrando el ratón.
-Asignar una categoría usando las teclas correspondientes.
-Navegar entre imágenes con las teclas indicadas.
-Finalizar la sesión para generar el archivo Excel con los recortes y metadatos.

SALIDA GENERADA

El programa produce:
-Un archivo Excel con la información completa de los recortes.
-Un conjunto de archivos de imagen que contienen cada recorte generado.
